File Commain
pwdPrint working directory.
cd [dir] Goto Home or specified directory.
cd - Goto previous directory.
ls -la --color=auto, ls -lrt, ls -lSr Quick dir listing, list files by date, list files by size
find dir [-maxdepth 1] [-type d] -name filename [| xargs cmd] Find files in dir and subdir by name, then exec cmd with the result.
grep [-i --color -H -n] [-r] string files [...] Find string in files, -r for subdir, -l just list file.
cp [-r] src dest Copy src to dest.
mv src dest Move src to dest or rename src to dest.
rm [-rf] filesdir Delete dir or file.
ln -s file link Create link for file.
cat/less/more file Output the content of file.
head/tail [-n number] file Output head or tail of file.
tail -f file Output the content of file as it grow.
touch file Create or update timastamp of file.
chmod number filesdir 4:read,2:write,1:exec,0:none, owner/group/world.
System
which command Show full path name of command.
date Print or change the operating system date and time.
who Determine the users logged on the machine.
last Show listing of users last logged-in on your system.
history Show the last commands executed.
set Show the current user environment.
uname -a Show kernel version and system architecture.
ps aux, top Print process status.
kill [-9] pid, killall procname Kill process.
free -m Print memory usage.
ps aux | awk '{print $4"\t"$2"\t"$11}' | sort -r Print memory usage for process.
df -H Print disk usage.
du -hs filesdir, du -s * | sort -k1,1rn | head Print disk usage for directory, -s equivalent to -d 0.
adduser username [-d dir] Create a new user using home dir.
Networking
ping hostPing host.
whois domainGet whois for domain.
dig [@dns-server] [any] domainGet DNS for domain.
lsof -i -P Show open port
lsof -p $$ List paths that process id has open.
lsof ~ List processes that have specified path open.
netstat -ano Show network status.
ifconfig -a Configure network interface parameters.
traceroute host Trace route to host.
ssh $USER@$HOST command Run command on $HOST as $USER.
ssh -gvN -L 8080:localhost:80 root@$HOSTForward connections to localhost:8080 out to $HOST:80.
scp -v -p -r -C [-P port] $USER@$HOST:/filedir . Copy remote filefir to local, and vice verse. Use -P to specify dest port.
curl [-o filename/-O] [--compressed] [--silent] "url"Transfer a URL, w/o -O or -o, the content will be print to stdout.
curl -I urlDisplay HTTP header.
curl -d "user=username&password=111" urlPost data.
curl -u user:password urlAuth for URL.
curl -r range URLRetrieve a byte range of URL content, 0--1024/--1024/1024-- first 1024, last 1024, 1024 to last.
curl -C - -O urlContinue/resume a previous file transfer.
curl -H "headername=headervalue" urlAdd header to request.
curl -L [--location-trusted] urlFollow redirect, and send auth data to all location.
wget -r --no-parent --level=1 --convert-links "url"Recursive retrieving url.

--backup-converted, Backup orig file before convert link.

-A '*.jpg', Just download jpg files.
wget --mirror "http://www.example.com/"Equivalent to -r -N --level=inf --no-remove-listing.
Archives and Compression
tar czvf file.tar files Tar file into file.tar
tar xzvf file.tar Untar into current dicectory
tar tf file.tar Show contents of archive
gizp [-g] file Compress or decompress file
Text manipulation
sed 's/string1/string2/g' Replace string1 with string2
sed -i "" 's/oldstr/newstr/g' `grep "oldstr" -rl .` Replace string1 with string2 in all files
sed '/ *#/d; /^ *$/d' Remove comments and blank lines
sed 's/[ \t]*$//' Remove trailing spaces from lines
sed -n '1000{p;q}' Print 1000th line
sed -n '10,20p;20q' Print lines 10 to 20
wc -l Count lines from stdout
