Git

Git Branch

git branch [-a]
List branch
git branch %branchName% [%otherbranch/tag%]
Create a local branch based on HEAD or other branch or tag
git branch -d %branchName%
git push %remoteRepoName% :%remoteBranchName%
Delete local or remote branch
git checkout %name%
Switch to branch
git checkout -b %name% [%otherbranch/tag%]
Equivalent to create new branch and switch to it.
git merge [--no-ff] %branchName%
git merge %remoteRepoName%/%remoteBranchName%
Merge specified branch (local or remote) to current branch
git fetch [%remoteRepoName%]
Fetch objects from remote repo, but doesn't merge
git pull [%remoteRepoName%]
Fetch and merge the tracking remote branch to local branch
git push
Push changes on current local branch to the tracking remote branch
git push %remoteRepo% %localBranch%:%remoteBranch%
git push [-u] %remoteRepo% %branchName%
Push changes on %localBranch% to remote, create remote branch when it isn't exist.
Equivalent to %localBranch% == %remoteBranch%
git checkout -b %newBranch% [%basedBranch%]
git push %remoteRepo% %newBranch%
Create a local branch and switch to it, then push the local branch to remote
git fetch [%remoteRepo%]
git checkout -b %branchName% %remoteRepo%/%branchName%
Fetch all remote changes, then create and switch a local branch based on the remote branch and tracking it.

Git Basic

git init
Create git repo
git clone repo_path_or_uri [-b branch] [target_dir]
Clone git repo to target_dir
git add $file
git add .
Add filesdir to the index
git -m "comments"
Commit with comments
git status
Show the working tree status
git diff $id1 $id2
Show changes between commits, commit and working tree
git log [--oneline --decorate] [-p]
Show commit logs
git reflog
Manage reflog information
git blame $file
Show modify details of each line of a file
git show $id/$id:$file
Show various types of objects
git name-rev --name-only %commitid%
Get tag by commit id
git branch --contains%commitid%
Get branch by commit id
git ``ls``-files
To see which files have been commited
git cherry-pick $id
Add the specified commit to current branch

Git Rollback

git checkout -- $file
Discard changes of tracked but unstaged files
git clean -df
Remove all untracked files, but doesn't touch the files ignored by git. Use -xdf to remove untracked+ignored files.
git clean -Xdf
Remove only files ignored by git, but keep manually created files.
git reset --soft/hard/mixed $id
Reset files to specified version.
--soft, doesn't touch the staged/unstaged/untracked files.
--mixed, and move staged files to original state(unstaged or untracked), doesn't touch the unstaged/untracked files.
--hard, also reset staged/unstaged files, doesn't touch the untracked files.
git commit -a --amend
Redo the last time commit, with new indexed file and comments

There are 4 status of the gited files, commited(git-commit), staged(git-add), unstaged(modified/deleted file), untracked(new file).

Git Remote

git remove [-v]
List remote repos
git remote show %remoteRepoName%
Show infomataion of remote repo
git remote add %remoteRepoName% %URL%
Add remote repo and specify a alias name for it
git remote rename %oldRepoName% %newRepoName%
Change the alias name of remote repo
git remote rm %remoteRepoName%
Forget remote repo
git remote prune %remoteRepoName%
Cleanup remote repo, used to cleanup deleted remote branch

Git Submodule

git submodule add %gitURI% %path/to/submodule%
Add submodule
git submodule update --init --recursive
Update submodules
git submodule rm %path/to/submodule%
git rm --cached %path/to/submodule%
Remote submodule.

当第一次更新或克隆submodule（非添加）时，submodule的代码会被拿下来，但是submodule的工作区没和任何branch绑定，因此需要先checkout到branch上，才能修改submodule，例如checkout master

Git Tag

git tag
git tag -l "name*"
List tags
git tag -a %name% [-m "comment"]
Create a tag with comments
git tag -a %name% %commitID%
Create a tag for specified commit
git tag -d %name%
Delete tag
git push %tagName%/--tags
Push specified tag or all tags to remote repo
